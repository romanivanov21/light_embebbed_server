Асинхронный многопоточный сервер на С++

Реализация асинхорнной модели обработки сетевых соединений.


Библиотеки:
  1) common - общие компоненты
  2) thread_api - обортка на std::thread, удовлетворяющий идеоме RAII и возможностью корректного прерывания
  3) network_api - библеотека работы сетью
